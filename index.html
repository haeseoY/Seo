1. 소인수분해

상세 개념

● 소수와 합성수1은 소수도 합성수도 아님. 2는 유일한 짝수인 소수.
● 소인수분해와 약수약수의 개수는 각 소인수의 지수에 1을 더해 모두 곱함.
● 최대공약수/최소공배수GCD: 나누어 주는 상황. LCM: 동시에 다시 만나는 상황.

실전 문제

기초: 12와 18의 최대공약수는?
정답: 6
공통 소인수 중 낮은 지수들의 곱 (2 × 3)
심화: 18 × x 가 어떤 자연수의 제곱이 되는 가장 작은 x?
정답: 2
18 = 2 × 3² 이므로, 지수가 홀수인 2를 보충해야 함.

2. 정수와 유리수

상세 개념

● 수의 분류유리수 = 정수(양의 정수, 0, 음의 정수) + 정수가 아닌 유리수
● 절댓값0과의 거리. 원점에서 멀수록 크며 부호를 제외한 값.
● 사칙연산 규칙음수(-)가 홀수개면 결과는 (-), 짝수개면 (+).

실전 문제

기초: 절댓값이 4인 두 수의 합은?
정답: 0
|+4|=4, |-4|=4 이므로 4 + (-4) = 0
심화: (-2)³ ÷ (4/5) - (-6) 계산
정답: -4
-8 × (5/4) + 6 = -10 + 6 = -4

3. 문자와 식

상세 개념

● 문자의 사용곱셈기호 생략, 숫자를 문자 앞에 배치, 1은 생략.
● 일차식의 계산동류항(문자와 차수가 동일)끼리만 덧셈, 뺄셈 가능.
● 일차방정식등식의 성질 활용. x항은 좌변, 상수항은 우변으로 이항.

실전 문제

기초: 3x + 5 = 11 의 해는?
정답: x = 2
3x = 6 이므로 x = 2
심화: 연속하는 세 정수의 합이 45일 때 가장 큰 수?
정답: 16
(x-1) + x + (x+1) = 45 → 3x = 45, x = 15. 따라서 큰 수는 16

4. 좌표평면과 그래프

상세 개념

● 좌표와 사분면제1(+,+), 제2(-,+), 제3(-,-), 제4(+,-). 축 위는 제외.
● 정비례 (y = ax)원점을 지나는 직선. x가 2배, 3배 되면 y도 2배, 3배 됨.
● 반비례 (y = a/x)한 쌍의 매끄러운 곡선. xy = a (일정)의 관계 성립.

실전 문제

기초: 점 (-2, 3)은 몇 사분면인가?
정답: 제2사분면
x가 음수, y가 양수인 영역.
심화: 반비례 y = 12/x 그래프 위의 점 중 x, y 좌표가 모두 정수인 점은?
정답: 12개
12의 약수 6개에 각각 ± 부호를 고려함.
